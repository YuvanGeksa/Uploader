## Cara Pakai

### 1) Siapkan GitHub Token
Buat **Personal Access Token** yang punya akses **write** ke repo tujuan.

### 2) Buka Website
Isi form:
- **GitHub Token**
- **Owner/Username**
- **Repository**
- Pilih **ZIP file**

Klik **Next** → pilih ZIP → **Upload Now**.

### 3) Cek Hasil
Setelah sukses, link repo/branch akan muncul di `>_ Activity Log`.

---

## Notes

- Jika repo kamu aktif (banyak perubahan), disarankan upload ke **branch baru** untuk menghindari konflik.
- ZIP yang terlalu besar / terlalu banyak file bisa bikin proses lebih lama.
